点操作符和中括号操作符的区别

在 JavaScript 中，点操作符（.）和中括号操作符（[]）都用于访问对象的属性。

点操作符（.）

语法：对象.属性名

特点：

  1. 访问的属性是静态属性，属性名必须是固定的，不能是变量或表达式
  2. 属性名是合法的标识符
    1. 不能以数字开头
    2. 不能包含空格、标点符号（下划线_和美元符$除外）
    3. 不能是 JavaScript 关键字（如if、function、class等）

中括号操作符（[]）

语法：obj[属性名]

特点：

  1. 访问的属性可以是动态属性，属性名可以是字符串、变量或表达式
  2. 可以访问包含特殊字符、空格或数字开头的属性名
