排查

原因

Flutter SDK 所在的 Git 仓库拉取更新时出错，导致 flutter doctor 检查 Git 状态失败。

  • 本地仓库和远程不一致
  • Git 被强制更新（forced update）
  • 权限 / 网络导致 Git 拉取标签失败
  • Flutter SDK 文件夹被占用、损坏

flutter doctor 会检查 Git 更新，所以直接抛出 exit code 1。
