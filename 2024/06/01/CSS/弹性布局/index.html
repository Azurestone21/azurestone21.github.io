弹性布局

弹性布局（Flexible Box Layout，简称 Flexbox）是 CSS3 引入的一种一维布局模型。

一、核心概念：容器与项目

Flex 容器（Flex Container）：通过给父元素设置 display: flex 或 display: inline-flex 激活 Flex 布局，该父元素即为 “容器”。

Flex 项目（Flex Item）：容器的直接子元素自动成为 “项目”（孙子元素不直接受容器控制），浮动、清除浮动、垂直居中属性（如 vertical-align）对项目无效。

二、容器的 6 个核心属性

flex-direction：控制项目的排列方向（主轴方向）

  1. row（默认）：水平从左到右
  2. row-reverse：水平从右到左
  3. column：垂直从上到下
  4. column-reverse：垂直从下到上

flex-wrap：控制项目是否换行

  1. nowrap（默认）：不换行，项目会压缩宽度适应容器
  2. wrap：自动换行，超出容器的项目到下一行
  3. wrap-reverse：自动反向换行（从下往上排）

flex-flow：flex-direction + flex-wrap 的简写

默认值为 row nowrap。

justify-content：控制项目在主轴上的对齐方式

  1. flex-start（默认）：项目靠主轴起点对齐 (左对齐)
  2. flex-end：项目靠主轴终点对齐 (右对齐)
  3. center：项目在主轴中间对齐 (水平居中)
  4. space-between：项目均匀分布：两端贴容器，中间间距相等
  5. space-around：项目均匀分布：项目两侧间距相等（边缘间距是中间的 1/2）
  6. space-evenly：项目均匀分布：项目之间及与边缘的间距完全相等

align-items：控制项目在交叉轴上的对齐方式

  1. stretch（默认）：项目未设置高度时，拉伸至与容器交叉轴高度一致
  2. flex-start：项目靠交叉轴起点对齐 (顶部对齐)
  3. flex-end：项目靠交叉轴终点对齐 (底部对齐)
  4. center：项目在交叉轴中间对齐 (垂直居中)
  5. baseline：项目按”文字基线”对齐(如不同字号文本对齐)

align-content：控制多行项目在交叉轴上的对齐方式

  1. stretch（默认）：多根主轴线拉伸至填满容器交叉轴高度
  2. flex-start：多根主轴线靠交叉轴起点对齐(顶部对齐）
  3. flex-end：多根主轴线靠交叉轴终点对齐(底部对齐)
  4. center：多根主轴线在交叉轴中间对齐 (垂直居中)
  5. space-between：多根主轴线均匀分布：两端贴容器，中间间距相等
  6. space-around：多根主轴线均匀分布：两侧间距相等

三、项目的 6 个核心属性

flex-grow：控制项目的 “放大比例”

当容器主轴方向有剩余空间时，项目根据 flex-grow 的数值分配剩余空间（默认值为 0，即不放大）。

flex-shrink：控制项目的 “缩小比例”

当容器主轴方向空间不足时，项目根据 flex-shrink 的数值压缩自身宽度（默认值为 1，即允许缩小）。

  • 数值为非负整数（0 表示不缩小，1、2… 表示缩小权重）；
  • 若所有项目 flex-shrink 之和为 n，则每个项目压缩的宽度比例为 自身flex-shrink / n。

flex-basis：控制项目在主轴上的 “初始尺寸”

定义项目在分配剩余空间前的初始宽度（主轴为水平）或高度（主轴为垂直），默认值为 auto（即项目自身内容宽度）。

  • 可设置具体数值（如 100px、20%）；
  • 若设置为 0，项目尺寸完全由 flex-grow/flex-shrink 决定；若设置为 auto，项目先保留自身尺寸，再分配剩余空间。

flex：flex-grow + flex-shrink + flex-basis 的简写

默认值为 0 1 auto。

1. 常见简写场景：
- flex: 1：等价于 flex: 1 1 0% → 项目均分剩余空间（常用作 “自适应宽度”）；
- flex: auto：等价于 flex: 1 1 auto → 项目先保留自身尺寸，再分配剩余空间；
- flex: none：等价于 flex: 0 0 auto → 项目不放大、不缩小，尺寸由自身决定（常用作 “固定宽度”）。

order：控制项目的排列顺序

默认情况下，项目按 “代码书写顺序” 排列（order: 0），通过设置 order 的整数数值（可正可负），可改变项目的排列优先级：

  • 数值越小，项目越靠前。
  • 若多个项目 order 相同，则按代码顺序排列。

align-self：控制单个项目在交叉轴上的对齐方式

覆盖容器的 align-items 属性，为单个项目设置特殊的交叉轴对齐方式。

  • 属性值与 align-items 完全一致（auto、stretch、flex-start、flex-end、center、baseline）。
  • 默认值为 auto（继承容器的 align-items）。
