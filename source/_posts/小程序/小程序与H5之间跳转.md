## 小程序跳转 H5

### webview

使用 webview（个人类型的小程序暂不支持使用）

```javascript
<web-view src={url} />
```

### 配置业务域名

在 MP 后台业务域名中添加需要跳转的 H5 页面域名（必须是 HTTPS 协议）。 确保域名已通过 ICP 备案。

## H5 跳转小程序

### URL Scheme

[https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/url-scheme.html](https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/url-scheme.html)

URL Scheme 分为两种：明文 URL Scheme 和 密文 URL Scheme。

1. **明文 URL Scheme**

**在「MP平台 - 左下角账号 - 账号设置 - 基本设置 - 隐私与安全 - 明文Scheme拉起此小程序」配置小程序路径**后，可自行根据如下格式拼接appid和path等参数，作为明文 URL Scheme 链接。

```plain
weixin://dl/business/?appid=*APPID*&path=*PATH*&query=*QUERY*&env_version=*ENV_VERSION*
```

- 参数说明
  - 【必填】APPID：需要打开的小程序 appid ；
  - 【必填】PATH：需要打开小程序的页面 path ，必须是已经发布的小程序存在的页面，不可携带 query；
  - 【选填】QUERY：通过明文 URL Scheme 打开小程序的 query ，最大512个字符，只支持数字，大小写英文以及部分特殊字符：!#$&'()\*+,/:;=?@-.\_~%`，需要url_encode；
  - 【选填】ENV_VERSION：要打开的小程序版本，正式版为release，体验版为trial，开发版为develop，仅在微信外打开时生效。注意：若不填写，则默认打开正式版小程序。

2. **密文 URL Scheme**

通过服务端接口可以获取打开小程序任意页面的加密 URL Scheme。

**跳转方式**

```javascript
// 方式1
window.location.href = schemelink;

// 方式2：点击链接
<a :href="schemelink">跳转至小程序</a>
```

### <font style="color:rgb(34, 34, 34);">JSSDK 接口</font>

webview <font style="color:rgb(34, 34, 34);">内嵌的 H5 可以使用 JSSDK 的接口返回，参数与小程序的接口一样</font>

```javascript
// 引入JSSDK
<script type="text/javascript" src="https://res.wx.qq.com/open/js/jweixin-1.3.2.js"></script>
// 或者安装JSSDK后使用
import wx from "weixin-js-sdk";

const jump = () => {
  wx.miniProgram.navigateTo({ ... })
  wx.miniProgram.navigateBack({ ... })
  wx.miniProgram.switchTab({ ... })
  wx.miniProgram.reLaunch({ ... })
  wx.miniProgram.redirectTo({ ... })
}
```

### 开放标签：<font style="color:rgb(34, 34, 34);">wx-open-launch-weapp</font>

[https://developers.weixin.qq.com/doc/service/guide/h5/opentag.html#%E8%B7%B3%E8%BD%AC%E5%B0%8F%E7%A8%8B%E5%BA%8F%EF%BC%9Awx-open-launch-weapp](https://developers.weixin.qq.com/doc/service/guide/h5/opentag.html#%E8%B7%B3%E8%BD%AC%E5%B0%8F%E7%A8%8B%E5%BA%8F%EF%BC%9Awx-open-launch-weapp)

**使用条件**

1. **<font style="color:rgb(34, 34, 34);">已认证的服务号</font>**<font style="color:rgb(34, 34, 34);">，服务号绑定“JS接口安全域名”下的网页可使用此标签跳转任意合法合规的小程序。</font>
2. <font style="color:rgb(34, 34, 34);">已认证的</font>**<font style="color:rgb(34, 34, 34);">非个人主体</font>**<font style="color:rgb(34, 34, 34);">的小程序，使用小程序云开发的</font>[<font style="color:rgb(87, 107, 149);">静态网站托管</font>](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/guide/staticstorage/introduction.html)<font style="color:rgb(34, 34, 34);">绑定的域名下的网页，可以使用此标签跳转任意合法合规的小程序。</font>

```javascript
<wx-open-launch-weapp
  id="launch-btn"
  username="小程序原始账号 ID（gh_ 开头的）"
  path="要跳转到的页面路径"
>
  <template>
    <button>打开小程序</button>
  </template>
</wx-open-launch-weapp>
```
