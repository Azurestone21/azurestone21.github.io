## 手机<font style="color:rgb(64, 64, 64);">底部安全区域（safe-area-inset-bottom）</font>

安全区域主要出现在 iPhone X 及以后的全面屏设备上，底部有一个小黑条，可能会遮挡页面底部的内容。为了解决这个问题，CSS提供了env()和constant()函数，用于设置安全区域与边界的距离。

### CSS兼容安全区域

安全区域相关的属性有 4 个：

- safe-area-inset-left：安全区域距离左边边界的距离
- safe-area-inset-right：安全区域距离右边边界的距离
- safe-area-inset-top：安全区域距离顶部边界的距离
- safe-area-inset-bottom ：安全距离底部边界的距离

可以使用`constant()`或者`env()`设置。

```css
padding-bottom: constant(safe-area-inset-bottom);
padding-bottom: env(safe-area-inset-bottom);
```

### JS判断是否有安全区域

[https://developers.weixin.qq.com/miniprogram/dev/api/base/system/wx.getWindowInfo.html](https://developers.weixin.qq.com/miniprogram/dev/api/base/system/wx.getWindowInfo.html)

```javascript
// 获取系统信息
const systemInfo = wx.getSystemInfoSync(); // 停止维护，建议使用 wx.getWindowInfo()
const systemInfo = wx.getWindowInfo();

// 判断是否有底部安全区域
const hasSafeArea =
  systemInfo.safeArea && systemInfo.safeArea.bottom < systemInfo.screenHeight;
```
