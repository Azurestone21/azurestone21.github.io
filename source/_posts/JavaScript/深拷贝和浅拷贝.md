## 深拷贝

深拷贝是递归拷贝深层次，属性为对象时，深拷贝是新开栈，两个对象指向不同的地址。修改一个对象的属性，不会改变另一个对象的属性

### 实现方法

- **JSON.parse(JSON.stringify())**
  - 用于通过序列化和反序列化创建对象的深拷贝。
  - 只适用于可以通过 JSON 序列化的对象，不能处理函数、正则表达式等非 JSON 可序列化的对象。
  - 无法正确处理循环引用的情况，会导致无限递归。
- **lodash.cloneDeep()**
  - Lodash 函数库中的一个函数，用于深度克隆对象或数组。
  - 可以深度复制对象、数组、嵌套的对象和数组，以及它们的属性。
  - 可以处理循环引用，避免了循环引用导致的无限递归问题。
  - 由于功能较为复杂，可能会在某些情况下比简单的 JSON 序列化和反序列化慢一些。
- **递归复制**

### 浅拷贝

浅拷贝，指的是创建新的数据，这个数据有着原始数据属性值的一份精确拷贝。如果属性是基本类型，拷贝的就是基本类型的值。如果属性是引用类型，拷贝的就是内存地址。两个对象指向同一个地址。

### 实现方法

- **Object.assign()**
- **扩展运算符**
- **Array.prototype.slice()**
- **Array.prototype.concat()**
