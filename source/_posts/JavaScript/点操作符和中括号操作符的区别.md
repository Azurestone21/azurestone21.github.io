在 JavaScript 中，点操作符（`.`）和中括号操作符（`[]`）都用于访问对象的属性。

```javascript
const obj = {
  name: "张三",
  age: 18,
  "user name": "小张",
  123: 456,
};
```

## 点操作符（`.`）

**语法**：`对象.属性名`

**特点**：

1. 访问的属性是静态属性，属性名必须是固定的，不能是变量或表达式
2. 属性名是合法的标识符
   1. 不能以数字开头
   2. 不能包含空格、标点符号（下划线\_和美元符$除外）
   3. 不能是 JavaScript 关键字（如if、function、class等）

```javascript
obj.name			// 张三
obj.age				// 18
obj.user name // 报错（包含空格）
obj.123 			// 报错（数字开头）
```

## 中括号操作符（`[]`）

**语法**：`obj[属性名]`

**特点**：

1. 访问的属性可以是动态属性，属性名可以是字符串、变量或表达式
2. 可以访问包含特殊字符、空格或数字开头的属性名

```javascript
obj["name"]; // 张三
obj["age"]; // 18
obj["user name"]; // 小张
obj[123]; // 456

const x = name;
obj[x]; // 张三（变量）
```

## 其他访问对象属性的方法

### 解构赋值

```javascript
const { name, age } = obj;
```

### Object.keys() / Object.values() / Object.entries()

```javascript
// 获取所有键并访问对应值
Object.keys(obj).forEach((key) => {
  console.log(obj[key]); // 依次输出 "张三"、18、"小张"、456
});

// 直接获取所有值
console.log(Object.values(obj)); // ["张三", 18, "小张", 456]

// 获取键值对并访问
Object.entries(obj).forEach(([key, value]) => {
  console.log(`${key}: ${value}`); // name: 张三; age: 18; user name: "小张"; 123: 456
});
```

### Reflect.get()

```javascript
Reflect.get(obj, "name");
```
