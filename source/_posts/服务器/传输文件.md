## 方案二：sftp

命令：`sftp root@ip`
回车后，会让输入密码，密码输入时不会显示出来

```sh
C:\Users\azure>sftp root@8.138.20.29
root@8.138.20.29's password:
Connected to 8.138.20.29.
sftp>
```

### 相关命令

- 查看 windows 上和 linux 上当前处于哪个目录

```sh
sftp> pwd
Remote working directory: /root
sftp> lpwd
Local working directory: c:\users\azure
```

- 如果不是目标目录，可以通过 `cd`（linux）和 `lcd`（windows）切换目录

```sh
sftp> cd /aaa
sftp> pwd
Remote working directory: /aaa
sftp> lcd D:\Project\Vue\Vue3\vue3-demo1\dist
sftp> lpwd
Local working directory: d:\project\vue\vue3\vue3-demo1\dist
```

- 通过 `ls`（linux）和 `lls`（windows）查看目录内是否有目标文件

![](https://azurestone21.oss-cn-guangzhou.aliyuncs.com/blogs/20260312001940955.png)

- 通过 `put` 从 windows 传输到 linux，通过 `get` 从 linux 下载到 windows（传文件夹需要用 -r）

![](https://azurestone21.oss-cn-guangzhou.aliyuncs.com/blogs/20260312001940956.png)

- 退出 `edit`

## 方案三：rz

优点：可以直接选择文件
缺点：不能上传文件夹

在xshell上执行

- 使用 `yum provides */rz` 这条命令，查看系统自带的软件包信息

![](https://azurestone21.oss-cn-guangzhou.aliyuncs.com/blogs/20260312001940953.png)

- 确认 rz 包存在之后，使用命令 `yum -y install lrzsz` 进行包的安装

![](https://azurestone21.oss-cn-guangzhou.aliyuncs.com/blogs/20260312001940954.png)

- 输入 `rz`，就可以选择本地文件进行上传

## 问题

> Permission denied (publickey,gssapi-keyex,gssapi-with-mic,password).

- xshell

```sh
vim /etc/ssh/sshd_config
```

- 修改

```config
PermitRootLogin: yes
PasswordAuthentication: yes
```

- 重启

```sh
systemctl restart sshd
```
