## 问题描述

容器显示有连续的中文、英文、数字，中文会自动换行，但是英文、数字只会在开头换行，后面不会再换行，并且会超出容器。

```html
<div>中文中文中文中文中文中文中文中文aaaaaaaaaaaaaaaaaaa111111111111111</div>
```

## 原因

**<font style="color:rgb(0, 0, 0) !important;">中文：无 “单词” 概念，单字有独立语义</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：中文最小语义单元是「单个汉字」（如 “开”“发”），即使拆分到不同行，也不影响阅读和理解（比如 “前端开发” 拆成 “前端”“开发”，语义仍清晰）。</font>

**<font style="color:rgb(0, 0, 0) !important;">英文 / 数字：“整体不可拆分” 是核心</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：英文的语义单元是「单词」（如 “frontend”“developer”），拆分单词会导致语义断裂；数字串（如手机号、ID “13800138000”）是连续的标识，拆分后会失去意义。</font>

<font style="color:rgb(0, 0, 0);">这也导致浏览器的「换行断点」规则不同：</font><font style="color:rgba(0, 0, 0, 0.85) !important;"></font>

| **<font style="color:rgb(0, 0, 0) !important;">文本类型</font>**       | **<font style="color:rgb(0, 0, 0) !important;">换行断点规则</font>**                                                                                                                                                                                                                                                                                                                                  |
| :--------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <font style="color:rgba(0, 0, 0, 0.85) !important;">中文</font>        | <font style="color:rgba(0, 0, 0, 0.85) !important;">每个汉字都是独立单元，</font>**<font style="color:rgb(0, 0, 0) !important;">任意两个汉字之间都允许作为断点</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">（无需额外空格），因此中文文本会自然在容器边缘拆分换行。</font>                                                                                                            |
| <font style="color:rgba(0, 0, 0, 0.85) !important;">英文 / 数字</font> | <font style="color:rgba(0, 0, 0, 0.85) !important;">以「空格」或「连字符（-）」作为唯一断点，</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">默认</font>\*\***<font style="color:rgb(0, 0, 0) !important;">单词 / 数字串内部不允许拆分</font>\*\*<font style="color:rgba(0, 0, 0, 0.85) !important;">（否则会破坏语义，比如 “developer” 拆成 “devel” 和 “oper” 会导致阅读困难）。</font> |

## 如何让英文、数字自动换行

### <font style="color:rgb(0, 0, 0);">方案 1：使用 </font>`<font style="color:rgb(0, 0, 0);">word-break: break-all</font>`<font style="color:rgb(0, 0, 0);">（优先推荐）</font>

允许在任意字符之间拆分换行（无论是否是单词内部、是否有空格）。

### 方案 2：使用 overflow-wrap: break-word（兼顾语义）

优先在 “合法断点”（空格 / 连字符）处换行，只有当无断点时，才在单词内部拆分（比 break-all 更保护语义）。
