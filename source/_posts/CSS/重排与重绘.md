---
title: "重排与重绘"
date: 2023-03-18 21:49:53
category: [CSS]

---

- **重排（Reflow）**：计算元素**几何属性**（布局）的过程。当元素的位置、大小等发生改变，影响了文档流（DOM树）的布局时，浏览器需要重新计算所有受影响元素的几何信息，这个过程开销**很大**。

- **重绘（Repaint）**：将元素**外观属性**（像素）绘制到屏幕上的过程。当元素的样式改变但不影响其布局时（例如只改变颜色、背景色、可见性等），浏览器会跳过重排，直接进行重绘。重绘的开销**相对较小**。

| **特性**     | **重排（Reflow / 回流）**                                                                                                                                                                                         | **重绘（Repaint）**                                                                                                                |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **触发原因** | **布局/几何属性**发生变化                                                                                                                                                                                         | **外观/绘制属性**发生变化，但不影响布局                                                                                            |
| **本质**     | 重新计算各元素的**尺寸和位置**                                                                                                                                                                                    | 将元素的**新样式**像素绘制到屏幕上                                                                                                 |
| **性能开销** | **高**，计算所有受影响元素的布局，是主要的性能瓶颈                                                                                                                                                                | **较低**，不涉及布局计算，只涉及绘制                                                                                               |
| **触发条件** | 1. 页面首次渲染 2. 浏览器窗口大小改变（`resize`事件） <br/> 3. 元素**尺寸、位置、内容**发生改变 <br/> 4. 添加或删除**可见的**DOM元素 <br/>5.  激活CSS伪类（如`:hover`）<br/> 6. 查询某些属性或调用某些方法（见下文） | 1. 修改**颜色、背景色、边框颜色、透明度**等 <br/> 2. 修改`visibility`、`outline`等属性 <br/> 3. 任何导致重排的操作也必然会导致重绘 |
| **常见属性** | `width`, `height`, `padding`, `margin`, `display`, `position`, `left`, `top`, `font-size`, `overflow` 等                                                                                                          | `color`, `background-color`, `border-color`, `visibility`, `text-decoration`, `background-image`, `outline` 等                     |
