## 什么是AI Agent？

AI Agent（人工智能智能体）是一种能够感知环境、进行决策并执行动作的自主软件系统。

## 为什么需要AI Agent？

了解为什么需要AI Agent前，我们需要先了解LLM（大语言模型）的缺点。

LLM虽然知识渊博，能回答各种问题，但是它只有“想”，没有“做”的能力。你告诉它想做些什么，它只能回答你操作步骤，不能执行任何操作。另外训练大模型的数据一般基于过去的数据，不能实时更新最新的信息。

```
用户: “北京今天天气怎么样？”
LLM: “我无法获取实时数据，建议你查看天气应用。”（或联网搜索后回答）
```

AI Agent的出现正是为了弥补LLM的这些缺点。它能够通过感知环境来获取最新的信息，并且能够调用外部API来执行各种操作来完成任务。

```
用户: “北京今天天气怎么样？如果下雨，帮我提醒带伞，并重新安排户外会议。”
Agent: 
  1. 调用天气API → 获取“有雨”
  2. 逻辑判断 → 触发提醒
  3. 访问日历API → 查找户外会议
  4. 发送通知 → 告知用户调整
```
