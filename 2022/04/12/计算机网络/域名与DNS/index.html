域名和 DNS

域名是什么

互联网中的每个设备都有一个唯一的编号来标识身份，这个编号我们称为 IP 地址（例如 IPv4 地址 203.0.113.1 或 IPv6 地址 2001:0db8:85a3:0000:0000:8a2e:0370:7334）。但是 IP 地址的组成比较复杂，难以记住，而域名就是为了解决这个问题而产生的。

简单来说，域名就是 IP 地址的名称，将复杂的 IP 地址转为容易记忆的名称。

域名必须和 IP 地址绑定，否则无法访问网址。

为什么要使用域名

  1. 便于记忆
  2. 品牌建设
  3. 灵活性：如果网站需要更换服务器（更换“建筑”），IP地址会改变，但域名可以保持不变。您只需要更新DNS记录，用户依然可以通过同一个域名访问您的网站。

域名的结构

域名由多个 “标签（Label）” 组成，标签之间用英文句点.分隔，且遵循从右到左的 “层级体系”，即最右侧是最高级，左侧依次为下级。

域名的层级（标签数量）理论上没有严格的 “最大限制”，但受DNS 协议规范和实际应用场景约束，通常最多支持 127 级，每个标签最多 63 个字符，完整域名（含所有句点）≤ 253 个字符。

实际场景中一般是以 3 级域名为主。过长的域名会导致可读性差，解析效率低。

以 www.baidu.com 为例：

  1. 顶级域名（TLD）：.com
  2. 二级域名：baidu
  3. 三级域名：www

DNS 是什么

DNS 是域名系统（Domain Name System），它的作用是将域名转为 IP 地址，从而找到对应的网络设备。

DNS 的工作流程

  1. 在浏览器地址栏输入域名www.baidu.com后。
  2. 计算机会联系 DNS（域名系统）。
  3. DNS 会查找 www.baidu.com 这个“名字”对应的真实 IP 地址。
  4. 浏览器拿到 IP 地址后，就能准确地找到百度网站的服务器。
  5. 服务器将网站的内容发送回浏览器，就看到百度的页面了。
